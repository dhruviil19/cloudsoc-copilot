# Demo Video Recording Checklist

Before recording:

- Start backend on port 8000
- Start dashboard on port 8501
- Reset database
- Use SOC Analyst role
- Make browser zoom around 90 percent
- Keep terminal visible only if needed

- Show dashboard overview
- Load advanced demo logs
- Run detection
- Show alert queue
- Show one incident detail
- Show MITRE ATT&CK View
- Show Attack Timeline
- Show Vulnerability Prioritization
- Export Markdown report
